# 📖 Overview

DeepShield is an **Agentic AI-powered Deepfake Detection and Authenticity Verification System** designed to detect manipulated video and audio content using **Multi-Modal AI Analysis**.

Unlike traditional deepfake detectors that simply classify media as *Real* or *Fake*, DeepShield autonomously analyzes multiple sources of evidence, verifies authenticity, performs intelligent reasoning, and takes appropriate actions such as generating alerts, flagging suspicious media, blocking malicious content, and producing verification reports.

# 🏛️ Technical Architecture

<p align="center">

</p>

### Architecture Layers

### 🔹 Input Layer
- Live Stream
- Video Upload
- Audio Upload
- Social Media Links

### 🔹 Frontend Layer
Interactive dashboard for media upload, monitoring, visualization, and reports.

### 🔹 Backend Layer
FastAPI-based inference engine responsible for preprocessing, AI inference, orchestration, and report generation.

### 🔹 AI/ML Layer
- Video Deepfake Detection (EfficientNet-B0)
- Audio Deepfake Detection (Wav2Vec2)
- Multi-Modal Fusion Engine
- Agentic Decision Engine

### 🔹 Storage Layer
Stores users, reports, logs, metadata, and inference history.

### 🔹 Response Layer
Autonomously performs:

- ✅ Allow
- 🚨 Alert
- 🚩 Flag
- ⛔ Block
